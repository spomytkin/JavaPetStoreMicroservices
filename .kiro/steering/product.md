# PetStore Product Overview

## Project Description
PetStore is a microservices-based e-commerce application for pet products. It's an educational project that demonstrates the evolution from a legacy JavaEE monolith to a modern microservices architecture.

## Purpose
- Serve as a practical example for Java developers, Full Stack Web App developers, team leads, and IT managers
- Demonstrate modern technologies, application design, and development techniques
- Showcase the transition from monolithic to microservices architecture

## Current Status
The project is in MVP phase with the following services implemented:
- Product Service (with AI assistant chat using langchain4j)
- Order Service
- Inventory Service
- Notification Service
- API Gateway using Spring Cloud Gateway MVC
- Frontend using Angular 18

## Current Focus
The team is currently implementing product classification capabilities to allow products to be categorized and organized hierarchically. This enhancement will enable better product organization, filtering, and discovery for both administrators and customers.

## Key Features
- Hierarchical category management
- Product-category associations
- Category-based product filtering
- REST API for category and product operations
- Data integrity and validation
